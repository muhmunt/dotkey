# Keyboard Shortcuts

Press **`?`** anywhere in the dashboard to open the shortcuts panel.

## Global

| Shortcut | Action |
|----------|--------|
| `⌘ K` / `Ctrl K` | Open global search |
| `?` | Open keyboard shortcuts |
| `Esc` | Close any open overlay |

## Global Search

| Shortcut | Action |
|----------|--------|
| `↑` `↓` | Navigate results |
| `Enter` | Go to selected result |
| `Esc` | Close search |

## Variable Table *(coming soon)*

| Shortcut | Action |
|----------|--------|
| `E` | Edit selected row |
| `C` | Copy value |
| `D` | Delete selected row |

## Tips

- The global search (`⌘K`) searches across **all projects** and **all variable keys** — useful when you have many projects and can't remember where a key lives.
- Shortcuts are disabled when your cursor is inside a text input or textarea.
- The shortcuts panel itself is also accessible from **Settings → Keyboard Shortcuts** in the user menu.
